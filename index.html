Tables

Reference

https://www.section508.va.gov/support/html/7_0.asp
https://www.w3.org/WAI/tutorials/tables/irregular/

Requirements:

Section 508 §1194.22 (g)
  • Each data table is indicated with a table tag.
  • Each row header and column header is identified with a th tag.
  • Each cell that's not part of a header is identified with a td tag.
  • Row and column headers are in the same table with table data cells. Note: Additional Data Table Requirements, for more information on this criteria.
  • Each defined row or column header has associated content.
  • When the table header is not the first cell in the row or column, the scope attribute is used and has the appropriate value of "row" or "col" (column).

Code Sample 508 Compliant Table 
        
<table>
   <caption>Two Row, Two Column Table</caption>
   <tr><th scope="col">Column 1</th>           <th scope="col">Column 2</th></tr>
   <tr><td scope="row">Row: 1, Column: 1</td>  <td>Row: 1, Column: 2</td></tr>
   <tr><td scope="row">Row: 2, Column: 1</td>  <td>Row: 2, Column: 2</td></tr>
   <tr><td scope="row">Row: 3, Column: 1</td>  <td>Row: 3, Column: 2</td></tr>
</table>

Summary If an image contains a logo or trademark, make sure it's accompanied by a brief description. It should not be described in detail. Summary A default language must be programmatically identified for each page, and for passages that use a language other than the default.
The language codes may be two-letter codes, such as en for English, four-letter codes such as en-US for American English, or other, less common, codes. The two-letter codes are defined in ISO 639-1, although modern best practice dictates that you should use the IANA subtag registry for your language code definitions Requirement

  • Setting the Language of a Page or Part of a Page
  • Assistive technology (AT) that supports multiple languages will be able to adjust to the pronunciation of the default natural language.
  • Braille translation software will be able to generate the correct characters based on the default language.
  • Setting the base language is especially important if various passages with different languages are present in the document.
Resource
  • link
  • The golden rule when creating language tags is to keep the tag as short as possible
Code Sample Set the base language of a page 

            
<html lang="us">...</html>

Set the language of specific parts of page

            
And with a certain <span lang="fr">je ne sais quoi</span>, 
she entered both the room, and his life, forever. 
<q>My name is Natasha,</q> she said. 
<q lang="it">Piacere,</q> he replied in impeccable Italian.

Requirement: Each list item must be tagged with HTML markup that can be understood by assistive technology (AT) and style sheets. Code Sample
  • Containing block size: percent ( % )
  • Viewport size: viewport units ( vw, vh, vmin, vmax )
  • Font height: ( em, rem )
  • Content size: ( auto, min-content, max-content )
  • Combination of the above: ( flex, min-width, max-width )
  • Avoid absolute units
